Implementing Predictive Analytics with Hadoop in Azure HDInsight

Learn how to implement predictive analytics solutions using Hadoop machine learning technologies like Mahout and Spark on Microsoft Azure HDInsight. In this four-week course, you will learn the basic concepts of data science and also use R programming on an HDInsight cluster. This course is the third in a series that explores big data and advanced analytics techniques with HDInsight; and builds on the concepts and skills learned in DAT202.1x: Processing Big Data with Hadoop in Azure HDInsight and DAT202.2x: Implementing Real-Time Analysis with Hadoop in Azure HDInsight. Note: To complete the hands-on elements in this course, you will require an Azure subscription and a Windows client computer.
